# Lessons

- When a helper agent introduces unexpected file changes, stop immediately and ask the user before proceeding.
- If the user asks for a clean reimplementation, overwrite target files from source-of-truth and re-run verification instead of relying on helper-generated artifacts.
- Keep SEO planning language out of visitor-facing UI copy; place keyword strategy only in internal docs (`tasks/seo-keyword-map.md`) and rewrite public text to be natural and user-benefit focused.
